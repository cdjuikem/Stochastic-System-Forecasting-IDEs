# Stochastic-System-Forecasting-IDEs 📈
### Numerical Simulation & Stability Analysis of Impulsive Differential Equations (IDEs)

## 📌 Project Overview
This repository contains a comprehensive suite of **MATLAB** scripts developed to simulate and analyze the dynamics of biological and environmental systems governed by **Impulsive Differential Equations**. The project focuses on modeling seasonal or human-induced interventions (impulses) and their long-term impact on system stability.

## 🛠 Key Features
- **Impulsive Dynamics Modeling:** Implementation of hybrid systems combining continuous ODEs with discrete-time jump processes.
- **Numerical Simulation:** High-precision solvers for linearised and non-linear dynamics (`impulsive_solution.m`, `continuous_dynamics.m`).
- **Threshold Analysis:** Automated computation of the Basic Reproduction Number ($R_0$) and stability regions.
- **Advanced Visualization:** Generation of publication-quality plots (Figures 2-9) to visualize complex system trajectories.

## 🧪 Technical Components
- **`R0.m`:** Script for computing stability thresholds in impulsive systems.
- **`linearised_dynamics.m`:** Implementation of local stability analysis around equilibrium points.
- **`predator.m`:** Application of the model to ecological predator-prey or pathogen-host interactions.

## 💻 Tech Stack
- **Language:** MATLAB
- **Methods:** Numerical Integration, Sensitivity Analysis, Impulsive Control Theory.
